# DESIGN.md — Project Cockpit
> Обязателен для КАЖДОГО UI-субагента. Отклонения — только через Team Lead.
> Направление: «приборная панель тёмной кабины» — Vercel-core минимализм + инструментальная точность. НЕ маркетинговый сайт.

## Дизайн-диалы (taste-skill)
- DESIGN_VARIANCE: 6 (асимметрия функциональная: канвас доминирует, панели подчинены)
- MOTION_INTENSITY: 6 (пружины, layout-транзишены, стаггер нод; без цирка)
- VISUAL_DENSITY: 6-7 (инструмент, не галерея; но читаемо для новичка)

## Палитра (Tailwind v4 @theme, только эти токены)
```css
@theme {
  --color-bg: #0B0E14;            /* фон приложения, НЕ чистый чёрный */
  --color-surface: #11151D;       /* панели */
  --color-surface-2: #161B26;     /* приподнятые элементы, ноды */
  --color-line: #232B3A;          /* hairline-границы */
  --color-line-strong: #33405A;
  --color-ink: #E6EAF2;        /* основной текст (класс text-ink) */
  --color-ink-dim: #8A94A7;    /* вторичный */
  --color-ink-faint: #5C6577;  /* подсказки/микрометки */
  --color-accent: #67E3F9;        /* cyan, desaturated — выбор/фокус/сквозные связи/AI */
  --color-accent-dim: #2AA7BF;
  --color-ok: #4ADE80;            /* статус стабильно */
  --color-warn: #FBBF24;          /* требует внимания */
  --color-risk: #F87171;          /* недоработки */
  --color-todo: #64748B;          /* задумано, не сделано (dashed) */
}
```
Правила: акцент ОДИН (cyan). Зелёный/жёлтый/красный/серый — ТОЛЬКО статус-семантика карты, никогда как декор. Никакого фиолетового, неона, outer-glow, градиентного текста. Тени — только тонированные под фон, редко.

## Типографика
- UI: **Geist** (variable, @fontsource-variable/geist). Заголовки: weight 500-600, tracking-tight, НЕ огромные (h1 = text-lg/xl в тулбаре).
- Данные/код/метки/цифры: **Geist Mono** (@fontsource-variable/geist-mono). ВСЕ числа — font-mono.
- Body 14px в панелях (инструмент), 16px в мастере/гиде-тексте. Line-height 1.5-1.6. Min 11px (микрометки, uppercase tracking-wide).

## Иконки
- Только `@phosphor-icons/react`, weight="regular" (1.5px stroke feel), размер 16/20. Никаких эмодзи нигде. Статус-точки — 6-8px круги токенных цветов (+ иконка/текст рядом, не только цвет).

## Раскладка (полноэкранная, desktop-first с деградацией до 375px)
- Каркас: сетка `grid-cols-[auto_1fr_auto]`: слева тонкий рейл линз (48px, иконки+тултипы), центр — канвас (доминанта), справа панель узла/гида 360-400px (сворачиваемая). Сверху бар 48px: проект-свитчер · крошки · поиск/действия.
- Панели НЕ карточки: surface + hairline `border-l border-line`, разделение `divide-y divide-line`, отступы p-4/p-5. Без rounded-контейнеров на всю панель; скругления только на интерактивных элементах (radius 8-10px).
- Канвас: фон bg с дот-сеткой React Flow (`gap=24 size=1` цвет line/40). Мобайл: панель узла становится bottom-sheet, рейл — нижняя навигация.

## Ноды карты (custom React Flow nodes)
- База: surface-2, border 1px line (hover: line-strong), radius 10px, padding 10-12px, min-w 150px. Заголовок 13px/500 Geist, подпись 11px text-dim mono.
- Статус: точка 7px слева от заголовка + при линзе «Риск» — заливка token/12% + border token/50%.
- todo-ноды: border-dashed, текст dim.
- Выбранная: border-accent + ring `0 0 0 3px accent/20%` (это фокус-ринг, не глоу).
- L0-блоки бизнес-контекста (маркетинг/продажи): приглушены (opacity 70%, без интерактива кроме тултипа «вне зоны разработки»).
- Drill-аффорданс: у нод с детьми — счётчик-чип «12» + CaretDown; hover: scale 1.01; active: scale-[0.98].

## Motion (пружины, не linear)
- Вход нод уровня: стаггер 30-40ms, spring stiffness 260 damping 26, y+8→0 + fade. Выход быстрее входа (~70%).
- Drill-down: fitView с duration ~500ms ease-out — ощущение «погружения»; крошка подсвечивается.
- Панель узла: слайд справа spring, interruptible. Линзы: кроссфейд содержимого канваса 150-200ms.
- Всё через transform/opacity. prefers-reduced-motion: анимации → fade 100ms.

## Состояния (обязательны)
- Loading: скелетоны формы контента (канвас: 5-6 серых нод-плейсхолдеров с шиммером), НЕ спиннер по центру.
- Empty (нет узлов уровня): композиция с иконкой + 1 фраза + действие.
- Error (Gemini без ключа/сбой): inline в панели гида, с recovery («вставить ключ», «повторить»).
- Кнопки: hover surface-подсветка, active scale-[0.98], disabled opacity-40 + cursor-not-allowed.

## Тексты интерфейса
- Русский, sentence case, без восклицаний, без «Elevate/seamless»-слопа. Кнопки — глагол: «Раскрыть ветку», «Спросить гида», «Собрать задачу».
- Демо-данные: реальные имена модулей Sales Bot из слепка; в мастере-примере — живой проект (не «Acme»).

## Anti-patterns (бан)
Эмодзи-иконки · фиолетовые/неон-градиенты · outer-glow · чистый #000 · Inter/системные шрифты · 3 равные карточки в ряд · спиннер на весь экран · немые иконки без label/тултипа · рандомные z-index (шкала: 0 канвас / 10 панели / 30 шапка / 50 оверлеи) · анимация width/height/top/left · серый-на-сером ниже 4.5:1 для основного текста.
